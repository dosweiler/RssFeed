      UniFi Access Point 6.5.64

      Source: UniFi Access Point/Switch/LTE | Releases | Ubiquiti Community | Date: 2023-07-18 |

      Overview UniFi Access Point 6.5.64 includes the improvements and bugfixes listed below. It's only available for Gen4 and Gen5 Access Points. Improvements [Gen6] Improved firmware update resiliency. Bugfixes Fixed incorrect neighbor reports after an RF Scan. [Gen2] Fixed the rare condition of low 5GHz throughput and high latency affecting certain APs (`TX OVERFLOW` in logs). [Gen2] Fixed high CPU usage when a lot of unauthorized guests are connected to a guest WLAN. [Gen2] Fixed 40MHz channel width not being applied correctly to 2.4GHz radio. [Gen4/Gen5] Fixed auth failing intermittently on WPA-Enterprise WLAN when PMKSA caching is disabled. [Gen4/Gen5/Gen6] Fixed kernel crashes to improve overall stability. [Gen6] Fixed Multicast Enhancement not working. This affects IPTV and other multicast-heavy traffic operations. [U6+/U6LR+] Fixed custom transmit power settings not being applied correctly. Additional information Refer to the Help Center article here to determine the UniFi Access Point generation. If you encounter any issues with this release, please provide us with some details on your setup and include the below information: AP model(s) Previous AP release version UniFi Network application version

      UniFi Access Point 6.5.62

      Source: UniFi Access Point/Switch/LTE | Releases | Ubiquiti Community | Date: 2023-07-18 |

      Overview UniFi Access Point 6.5.62 includes the improvements and bugfixes listed below. It's only available for Gen2, Gen3 and Gen6 Access Points. Improvements Improve device adoption and removing resiliency. [Gen6] Improved firmware update resiliency. [U6-Enterprise] Added 6GHz support for Canada. Bugfixes Fixed kernel crashes to improve stability. Fixed devices occasionally failing to factory reset when forgotten. [Gen2/Gen3] Fixed 5GHz radio getting stuck if a DFS radar event occurs during the CAC timeout. [Gen2/Gen4/Gen5] Fixed kernel crashes to improve stability. [Gen2/Gen3/Gen6] Fixed NAS-IPv6-Address being sent instead of NAS-IP-Address (v4) when the RADIUS server is reachable over IPv6. [Gen4/Gen5] Fixed kernel crashes to improve overall stability. [Gen4/Gen5] Fix first client connection after boot failing occasionally. [U6+/U6LR+] Fixed TFTP recovery not working sometimes, causing the AP to boot with the preexisting firmware instead of the newly flashed version. [U6+/U6LR+] Fixed 5GHz radio stuck in a DFS Waiting state if no 2.4GHz WiFi network is assigned to the AP. [U6-Lite/U6-LR] Fixed clients getting disconnected intermittently, leading to clients bouncing between 2.4GHz and 5GHz. [U6-Lite/U6-LR] Fixed 5GHz channel occasionally not broadcasting after a channel change. [U6-Lite/U6-LR] Fixed low data rates being used for some 5GHz clients after switching to a DFS channel. Additional information Refer to the Help Center article here to determine the UniFi Access Point generation. If you encounter any issues with this release, please provide us with some details on your setup and include the below information: AP model(s) Previous AP release version UniFi Network application version
